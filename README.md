# espcam-breakout-board
high-end ESP32-Cam breakout-board for IOT and monetering systems , so there has been a problem with the espcam and its the programming because it does not have a USB to UART converter so i am making a breakout board for it that solves that problem and adds more features like :tp5100 for charging ,MT3608 for 5v,3.3v,output, bme280 for IOT  also an IMU mpu6000 ,BMS( DW01A protection IC with 2 mosfets), 
it supports 4.2v batteries and cells any higher voltage will cut the power because the BMS is for 1s configuration, the charging is at 2A, bme temperature accuracy is ±0.5°C and humidity ±3%  Barometric Pressure ±1.0 hPa its set to i2c and its address is 0x76 or 0x77, mpu6000 is switched to i2c (cs pin is high) and address 0x69 accuracy Gyro: ±250 to ±2000 °/s accelerometer ±2g to ±16g internal tempurature sensor ±1°C
